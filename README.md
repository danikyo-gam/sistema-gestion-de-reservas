# Sistema de Gestión de Reservas

## Descripción

Sistema de gestión de reservas desarrollado en Python.  
Permite crear, editar y consultar reservas mediante un sistema de almacenamiento basado en archivos planos.

El proyecto está ambientado en un entorno de fantasía, incorporando una tematización inspirada en el video juego: La Leyenda de Zelda. franquisia que es propiedad de Nintendo
Y esta siendo usada solo con fines academicos y de aprendizaje

---

## Funcionalidades

- Crear reservas
- Editar reservas
- Consultar reservas existentes
- Almacenamiento persistente mediante archivos
- Interfaz de consola interactiva
-El admnistrador del programa (datos en el archivo admins.txt) tiene las funcionalidades de:
- Crear usuario
- Agregar Vuelo
- Editar Vuelo
- Ver estadisticas 
---

## Tecnologías utilizadas

- Python 3
- Manejo de archivos
- Estructuras de control
- Funciones y modularización
